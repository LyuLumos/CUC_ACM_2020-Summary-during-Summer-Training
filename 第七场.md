
### [B.   Mask Allocation ](https://ac.nowcoder.com/acm/contest/5672/B)  

### **题意**  
将总数为n*m的口罩，分成m份n个口罩和n份m个口罩，问最少需要多少个箱子（每个箱子不可以再拆分）

### **思路**  
取n和m中小的那个数为m，初始状态为m个装有n个口罩的箱子，保留m个有m个口罩的箱子，然后将剩下的n-m个包含m个口罩的箱子拆分，每次都尽可能的拆分成最大的数来将m个m个口罩的箱子补齐为n个m个口罩的箱子。

### **代码**  
```  
#include<bits/stdc++.h>
typedef long long ll;
using namespace std;
vector<int>ans;
void solve(int n,int m){
    if(m==0) return;
    for(int i=0;i<(n/m)*m;i++) ans.push_back(m);
    solve(max((n%m),m),min((n%m),m));
}
int main()
{
    int t,n,m;
    scanf("%d",&t);
    while(t--){
        scanf("%d%d",&n,&m);
        ans.clear();
        solve(max(n,m),min(n,m));
        printf("%d\n",ans.size());
        for(auto i:ans) printf("%d ",i);
        printf("\n");
    }
	return 0;
}
```  






### [D.   Fake News ](https://ac.nowcoder.com/acm/contest/5672/D)


### **题意**  
判断$1^2+2^2+...+n^2$的和是否是一个平方数
  

### **思路**  

法一

只有当n=1和24的时候，才是平方数。

法二

利用$1^2+2^2+...+n^2=\frac{n(n+1)(2n+1)}{6}$的公式，分别判断n是否为6的倍数，以及n/6,n+1,2n+1是否为平方数，如果是则和为平方数。


### **代码**  
```  
#include<stdio.h>
#include<math.h>
using namespace std;
typedef long long ll;
bool fun(int x)
{
	if((int)sqrt(x)*(int)sqrt(x)==x)return true;
	else return false;
}
int main() 
{
	int t;
	scanf("%d",&t);
	while(t--)
	{
		ll n;
		ll s=0;
		scanf("%lld",&n);
		if(n==1)printf("Fake news!\n");
		else if(n%6==0&&fun(n/6))
		{
			if(fun(n+1)&&fun(2*n+1))printf("Fake news!\n");
			else printf("Nobody knows it better than me!\n");
		}
		else printf("Nobody knows it better than me!\n");
	}
	return 0;
}
```  
